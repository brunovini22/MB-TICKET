### 📋 Sobre o Projeto
O **MB-TICKET** é uma solução completa para a venda de ingressos e gestão de eventos online. A plataforma foi projetada para suportar alta demanda, oferecendo uma experiência de compra rápida, segura e intuitiva para o usuário final, além de um painel robusto para organizadores de eventos.

Este projeto une design moderno com uma arquitetura técnica rigorosa, preparada para o mercado de entretenimento.

---

### 🛠️ Stack Técnica & Performance
O ecossistema do projeto foi escolhido para garantir estabilidade durante picos de acesso (abertura de vendas):

- **Frontend:** React + TypeScript (Interface reativa e segura).
- **Build Tool:** Vite + Bun (Otimização máxima de entrega de pacotes).
- **Testes de Stress:** Vitest (Garantindo que o fluxo de compra não falhe).
- **Estilização:** Tailwind CSS (Layout focado em conversão e Mobile First).
- **Qualidade:** ESLint configurado para manter o padrão de código limpo.

---

### ✨ Funcionalidades da Plataforma
* **Discovery de Eventos:** Filtros inteligentes por categoria, data e localização.
- **Checkout Fluido:** Fluxo de reserva e compra otimizado para reduzir o abandono de carrinho.
- **Gestão de Lotes:** Controle dinâmico de disponibilidade de ingressos.
- **Validação:** Estrutura preparada para integração com QR Codes de acesso.
